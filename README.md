# acestep.cpp

Minimal CUDA/C++17 implementation of the ACE-Step 1.5 music generation inference backend.

## Acknowledgements

This project is an independent implementation based on the original ACE-Step 1.5 by ACE Studio and StepFun.
All model weights are theirs, this is just a native backend.
